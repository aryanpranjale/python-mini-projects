# Rock Paper Scissors (Python)

A terminal-based Rock Paper Scissors game built in Python as part of my Python learning journey. I expanded the original course project by adding my own features to make it feel like a complete game instead of a single-round demo.

## Features

* Choose the number of rounds to play (defaults to 3 if Enter is pressed)
* Input validation for all user inputs
* Live score tracking after every round
* ASCII art for Rock, Paper, and Scissors
* Random computer moves
* Organized into multiple functions for better readability and maintainability

## Planned Features

* Random trash talk for wins and losses
* Sudden death mode if the scores are tied after all rounds
* Play Again option
* More detailed game statistics

## Concepts Practiced

* Functions
* Loops
* Conditional statements (`if`, `elif`, `else`)
* Lists
* User input validation
* `try` / `except`
* Returning values from functions
* Multiple return values (tuple unpacking)
* Random module (`random.choice`)
* Variables and score management

## How to Run

1. Make sure Python 3 is installed.
2. Clone or download this repository.
3. Run the program:

```bash
python rock_paper_scissors.py
```

## What I Learned

This project taught me much more than just building a Rock Paper Scissors game. While developing it, I learned how to break a program into smaller functions, validate user input, keep track of data across multiple rounds, return values from functions, and think about writing cleaner, more organized code. I also started focusing on making the game more enjoyable by adding my own features instead of only following the tutorial.

This is Version 1 of the project, and I plan to revisit it in the future as I learn more Python and improve its design.
