# Caesar Cipher

## Description

A Python implementation of the Caesar Cipher encryption technique. The program allows users to encrypt and decrypt messages by shifting letters in the alphabet by a specified number of positions.

## Features

* Encrypt text using a custom shift value
* Decrypt encrypted text
* Preserves spaces and punctuation
* User-friendly command line interface

## Concepts Used

* Functions
* Loops
* Strings
* User Input
* Conditional Statements

## How to Run

```bash
python caesar_cipher.py
```
