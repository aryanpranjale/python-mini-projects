
# Russian Roulette Simulator

## Description

A command-line Python game inspired by Russian Roulette. The program randomly determines whether the player survives each round or loses. As the game progresses, the danger increases, making survival less likely.

## Features

* Randomized chamber generation
* Interactive gameplay
* Increasing difficulty after each successful round
* Dramatic game-over messages
* Exception handling demonstration

## Concepts Used

* Functions
* Loops
* Lists
* Random Module
* User Input
* Exception Handling

## How to Run

```bash
python russian_roulette.py
```

## Example Gameplay

```text
Welcome to Russian Roulette!

Round with 6 chambers. Press Enter to pull the trigger...

Click. You're safe... for now.

Round with 7 chambers. Press Enter to pull the trigger...

Bang! You lost. The bullet was in the chamber.
Error: C:\ has been corrupted. System failure!
```
