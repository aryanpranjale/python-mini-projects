
# Luhn Algorithm Card Validator

## Description

A Python program that validates card numbers using the Luhn Algorithm. The program removes spaces and hyphens from user input and checks whether the card number passes the Luhn checksum test.

## Features

* Accepts card numbers containing spaces or hyphens
* Uses the Luhn Algorithm checksum calculation
* Displays whether the card number is valid or invalid

## Concepts Used

* Functions
* String Manipulation
* Loops
* Slicing
* Mathematical Operations

## Current Limitations

* The program expects numeric input after spaces and hyphens are removed.
* Input validation for alphabetic characters has not been implemented yet.
